擊球問題

擊球問題:

coppeliaSim模擬:版本0

最初的版本一使用coppeliaSim模擬球無法飛遠或高。

問題修改-1

coppeliaSim模擬:版本1

繪圖人員修改了擊球棒尺寸，使用coppeliaSim模擬球可以飛高。

問題解決:

coppeliaSim模擬:版本2

繪圖人員調整及球棒前端的角度，使用coppeliaSim模擬球可以飛遠且有一定的高度。

版本2 擊球點圖片

coppeliaSim 比例問題

問題:

coppeliaSim模擬:版本3

經過版本2的計算後的距離，繪圖人員新增籃框與調整距離，但是使用coppeliaSim模擬球擊不遠。 

問題討論:

以下是2個版本的stl檔，打開檔案會發現2個檔案大廈會差不多，但是放到coppeliaSim裡，版本2比例會自動放大。

第二版本stl檔

第三版本stl檔

以下是版本2與版本3使用參數一樣模擬的影片

解決版本:

coppeliaSim模擬:版本4

因為版本2在coppeliaSim裡被自動放大10倍，所以版本3無法擊中籃框，所以版本4放大10後就能擊中籃框。

結論:

沒想到coppeliaSim的比例不一樣，模擬會差很多，我猜想是因為軸放大10倍，導致紐力矩也放大10倍，因為力量不一樣，所以距離有差。

stage2-ag12 python keyboard

使用python keyboard 來操控投籃機

程式碼:

import sim as vrep
import sys
import keyboard
# child threaded script: 
#simExtRemoteApiStart(19999)
 
vrep.simxFinish(-1)
 
clientID = vrep.simxStart('127.0.0.1', 19999, True, True, 5000, 5)
errorCode,left_motor_handle=vrep.simxGetObjectHandle(clientID,'Revolute_joint',vrep.simx_opmode_oneshot_wait)

if clientID!= -1:
    print("Connected to remote server")
    while True:
        try:
            if keyboard.is_pressed('up'):
                  vrep.simxSetJointTargetVelocity(clientID,left_motor_handle,-16, vrep.simx_opmode_oneshot_wait)
                  print('up ') 
            if keyboard.is_pressed('down'):     
                  vrep.simxSetJointTargetVelocity(clientID,left_motor_handle,1, vrep.simx_opmode_oneshot_wait)
                  print('down')
            if keyboard.is_pressed('space'):
                  vrep.simxSetJointTargetVelocity(clientID,left_motor_handle,0, vrep.simx_opmode_oneshot_wait)
                  print('space')
                  
            else:
                pass
        except:
            break
else: 
    print('Connection not successful')
    sys.exit('Could not connect')
 
 
  
 
if errorCode == -1:  
    print('Can not find left or right motor')
    sys.exit()

Bug

Wsgi.py 打不開

問題:

新創小組倉儲，想要打開動態網頁出現以下問題

我的解法如下，底下有正確解。

解決方法:

1.把倉儲裡的localhost.crt', 'localhost.key 檔案要拉到cmsimde資料夾裡面

2.執行python wsgi.py，即可解決問題

正確解:

因為老師想要在倉儲裡直接打開網站和上傳資料，所以在cmsimde目錄下打開網頁會出問題。

直接在倉儲目錄裡打cms可打開網頁，用acp即可上傳至github。
